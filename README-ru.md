# **Google Gemini бесплатно в Google Colab!**
## **AI Pie 2.0 - самодельный чат с Gemini**
Я зашёл в Google Colab на их приветственную страницу, и случайно наткулся на бесплатный API Gemini без ключа. Я сразу же этим воспользовался и ради интереса написал эту программу.

**Программа поддерживает:**
  - Команды
  - Историю сообщений
  - Автоматическое удаление старых сообщений

**Абсолютно бесплатно!**

Перейдите в раздел Чат и запустите ячейку. (наведите курсор и нажмите кноку запуска)
## **Инструкция**
### **Команды**
`/exit` - выход из программы

`/clear` - очистка истории сообщений с экрана пользователя и из памяти ИИ

`/clear_screen` - очистка истории сообщение с экрана пользователя
